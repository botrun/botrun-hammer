## 地雷與經驗
- 「啟動 botrun-hammer」不等於啟動 Hammerspoon app 本身，而是重新載入 Hammerspoon 設定讓 `~/.hammerspoon/botrun-hammer.lua` 生效
- 使用 `hammerspoon://reload` URL scheme 比手動點選 menu bar 更快，且可在 script 中自動化
- 先用 `pgrep -x Hammerspoon` 確認程序存在，避免對尚未啟動的 app 發送 reload URL
- 若 Hammerspoon 未運行，應先 `open -a Hammerspoon` 再 reload
