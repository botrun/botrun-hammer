# 波特槌 🔨 五分鐘學會用嘴巴寫程式

---

## Slide 1：你有沒有打字打到手痠？

### 畫面
- 一張打字打到崩潰的梗圖

### 講稿

> 大家好！
>
> 問你一個問題：你一天花多少時間在打字？
>
> 寫 code 要打字、下 prompt 要打字、回訊息要打字。
> 有時候腦袋想得很快，但手就是跟不上。
>
> 今天教大家一招：**用嘴巴打字**。
> 你只要會說話，就會用。

---

## Slide 2：看我示範一次

### 畫面
- 螢幕錄影 GIF，三步驟動畫

### 講稿

> 來，看我操作一次：
>
> 第一步，按 **fn+F5**。螢幕會出現「正在傾聽」。
>
> 第二步，**說話**。想說什麼就說什麼。
>
> 第三步，再按一次 **fn+F5**。等幾秒，字就自動打好了。
>
> 就這麼簡單。三個步驟。一個按鍵。
>
> 重點是：**你在哪個 App 都能用**。
> Claude Code、VS Code、Line、Notion、Google Docs，通通可以。
>
> 另外兩個小功能：
> - **fn+F6**：看你之前說過什麼（最近 30 筆），選一個直接複製
> - **fn+F7**：找你之前的錄音檔

---

## Slide 3：怎麼裝？一行搞定

### 畫面
- 大大的終端機，裡面就一行指令

### 講稿

> 裝這個東西超簡單。
>
> 打開終端機，貼這一行：
>
> ```
> curl -fsSL https://raw.githubusercontent.com/botrun/botrun-hammer/main/install.sh | bash
> ```
>
> 它會自己裝好所有需要的東西，你什麼都不用管。
>
> 唯一要準備的：去 Google 申請一個免費的 Gemini API Key。
> 網址是 **aistudio.google.com/apikey**，有 Google 帳號就能申請。
>
> 安裝的時候它會問你 Key，貼上去就好了。
>
> **不用錢、不用 GitHub 帳號、不用密碼。**

---

## Slide 4：它會不會掛掉？不會。

### 畫面
- 簡單流程圖：說話 → Gemini 轉文字 → 掛了？自動切備案

### 講稿

> 你可能會問：API 掛了怎麼辦？
>
> 放心，它有備案機制：
> - Gemini 掛了 → 自動重試一次
> - 重試還是掛 → 自動切換到國網中心的 API
> - 你完全不用管，它自己處理
>
> 還有幾個貼心的設計：
> - **自動跳過假麥克風** — 如果你有裝 Teams 或 Zoom，它不會錄到虛擬裝置
> - **自動更新** — 有新版本它自己會升級
> - **簡體自動轉繁體** — 不用再手動改
